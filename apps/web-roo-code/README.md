# @roo-code/web-roo-code

Marketing website for Roo Code - An AI coding assistant that runs in VSCode and the terminal.

## Overview

This is the official marketing site built with Next.js 15, featuring:

- Product information and features
- Documentation and guides
- Evaluation results display
- Community resources

## Development

```bash
# Install dependencies from monorepo root
pnpm install

# Start development server
pnpm --filter @roo-code/web-roo-code dev

# Build for production
pnpm --filter @roo-code/web-roo-code build

# Start production server
pnpm --filter @roo-code/web-roo-code start
```

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Animations**: Framer Motion
- **Analytics**: PostHog
